Cepheid Visualizer

Project About

Why this project exists

The visualizer connects sound and visual expression in a compact Processing sketch. The project was inspired by analog oscilators. This project aims to capture the aesthetic using screen-space shaders and a set of audio-reactive primitives that are practical for realtime use.

Design goals

  • Keep the interface simple and immediate for live performance
  • Use GLSL where it helps reduce CPU load and to implement bloom and distortion effects
  • Provide clear, commented code so others can learn and modify it

Who I am

I perform and compose under the name Mitsune The Maid. I make music that blends shoegaze, post-punk and electronic elements. I like to integrate visuals with sound for live shows and recordings. This repository is intended both as a personal tool and as a learning resource for others.
