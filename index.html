Cepheid Visualizer

Project About

Audio reactive visuals in Processing

Cepheid Visualizer is a Processing sketch combining two audio reactive modes. Mode 0 is a PBR scene with bloom and mini spheres that react to sound. Mode 1 is a lightweight black hole visualization with six audio-driven rays and a screen-space distortion shader.

Download ZIP View on GitHub

Quick start

  1. Open the sketch folder in Processing.
  2. Install the Processing sound library if you do not have it.
  3. Run the sketch. Use keys 1 and 2 to switch modes. Move mouse to move the black hole.

Files included

  • GenerationsCombined.pde
  • MiniSphere.pde
  • data/pbr.vert.glsl
  • data/pbr.frag.glsl
  • data/generations.frag
  • data/glowFrag.glsl

Controls

  • 1: Mode 0 PBR
  • 2: Mode 1 Black hole
  • Mouse: Move black hole
  • + / - or arrows: change black hole radius
  • f: toggle FFT mode

Technical details

The sketch uses Processing with the sound library and several GLSL shaders. The heavy visual work is done in fragment shaders so the CPU work remains minimal. The black hole effect is a screen-space distortion that simulates lensing and a glowing accretion ring.

Dependencies

  • Processing 3.5 or later
  • processing.sound library
