Cepheid Visualizer

About Download

Standalone audio reactive visualizer

A small, exported Processing application with multiple real-time visual modes, bloom and persistence trails. The app uses your microphone input and is ready to run — no Processing or extra libraries needed.

Features

  • 6 visualization modes (keyboard control)
  • Audio-reactive (FFT spectrum + RMS loudness)
  • Bloom and persistence trails
  • Dynamic morphing in advanced modes

Quick start

  1. Download and run the exported application (no Processing needed)
  2. Allow microphone access if requested — the app is audio-driven
  3. Switch visual modes with keys 0–5
Controls
  • 0–5 — switch visual mode
  • + / - — adjust persistence (trail length)
  • B — toggle bloom
  • F — toggle FFT processing
  • Arrow keys — rotate camera

Note: modes 7 and 8 are not exposed in the public build.
